# DR Landing Page — Deright Precision Machinery

Premium dark-themed single-page landing site for Deright Precision Machinery Co., Ltd. (德瑞精密機械有限公司). Features advanced scroll-driven particle dynamics (where the "DR" logo letters float, scatter on hover, and disperse on scroll) and custom microscope-style magnifier lens.

## 🔗 Live Demo
You can view the live site here:
**[https://juniperlin.github.io/DR_Landing_Page/](https://juniperlin.github.io/DR_Landing_Page/)**

*Note: If you just created this repository, make sure to enable GitHub Pages in your repository settings (Settings -> Pages -> Build and deployment -> Source: Deploy from a branch -> Select `main` branch, `/root` folder, and click Save).*

---

## 🛠️ Features
- **Responsive Layout**: Designed for seamless viewing across desktops, tablets, and mobile devices.
- **Scroll-driven Particle Dynamics**: Particles slowly assemble into the "DR" letters, scatter organically on hover via the custom cursor, disperse/spread outward when scrolling down, and re-gather back into letters when scrolling to the top.
- **Microscope Magnifier Cursor**: Displays a neon-yellow custom circle replacing the system cursor on the hero section. Inside the circle, particles are magnified 100× to represent fine spray dryer powder particles.
- **Conditional Cursor Visibility**: The custom magnifier hides and the standard browser pointer is restored automatically as you scroll down to interact with content.
- **Modern Design System**: Features a dark, premium aesthetic with subtle glassmorphic container panels.


